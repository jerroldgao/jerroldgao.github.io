微服务的扫地僧--Dapr中文v1.5发布！

发表于2022-03-19|更新于2022-03-19|技术
|字数总计:576|阅读时长:1分钟|阅读量:

喜讯！

经过Dapr 中文文库译制小组的努力，Dapr v1.5的大部分文档翻译终于完成了！各位对云原生和微服务框架感兴趣的童鞋一定不能错过这个分布式应用程序运行时。我们翻译小组也是希望通过对Dapr文档中文的支持，让Dapr在华语的云原生社区更好地推广开来，欢迎各位使用Dapr，参与到Dapr开源社区的各种活动中来。

简介

Dapr 目前是 CNCF 的孵化级项目，是一个以应用开发为核心的分布式应用进行时。通过它，您能进行跨语言，跨框架的开发，甚至你可以试着把 Java 的 Spring 和 Golang 的 Kratos 在同一框架下进行开发，使得语言、框架不再变得你死我活。并且该框架可以在自托管或是K8s支持下部署，使得平台、环境不再成为障碍。有别于 Service Mesh 注重网络，Dapr 更注重应用本身，服务间通讯和可观测性，并且原生支持了强大的 Actor 并发模型。Dapr 还对类似像 Kafka，Redis 以及各大云厂商的中间件进行了整合，使得开发能更注重业务本身。想了解更多，请阅读 Dapr 文档。

我自己很喜欢把Dapr比喻做跨语言的 Spring boot，但是事实上 Dapr 目前能做的事情和愿景比 Spring boot 还要更大。Dapr 跨语言框架平台的能力，使得其像是只手让萧远山、慕容博和解的藏经阁老僧，让各位绝世高手达到和解。不过Dapr 对中间件的支持上目前没有 Spring boot 这么完整，而 Dapr 还在迅速地迭代成长，相信未来某一天 Dapr 会成为云原生框架这个王冠上最璀璨的宝石之一。

翻译进度

目前还有部分的开发应用和接近90%参考资料没有翻译完全。接下来也是我们翻译v1.6版本的重点。如果各位发现翻译中的勘误或者想加入 Dapr 中文翻译小组，欢迎联系我或是通过搜索「Dapr 中文文库译制小组」来加入我们。
